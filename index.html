Aeon PHP Project 2.0 - Silvergate Framework

Silvergate framework is an easy-to-use, object-oriented, MVC-based, and open source PHP framework that is developed under ION Developer Group. It redesign the first framework developed by IDG named Amuttal, to improve the maintainability, speeds, and the design.

To use this framework, simply download it in TAR or ZIP format. Then, extract it to your web server root folder (e.g htdocs on Apache).

You can also fork this repository to help us developing this framework. We developed this framework using Netbeans IDE 7.0.1

The leader of this framework development on ION Developer Group is Aditya Aida Purwa.
